Create an agent with Podman
Supported in:
Google secops
SOAR
Navigate to
SOAR Settings
>
Advanced
>
Remote Agents
.
Click
add
Add Agent
to start the installation process.
Select the
Podman
option.
In the first step, name the agent and choose the environment where it will 
    be installed.
Follow the remaining steps in the platform wizard, including running
  the generated Podman command on your local machine.
After the deployment finishes, click
Next
to run a connectivity check 
    with your new agent. A success notification will confirm the connection. If 
    the check fails, a failure notification will display with helpful 
    troubleshooting links.
Click
Done
. The remote agent is now added to the platform.
