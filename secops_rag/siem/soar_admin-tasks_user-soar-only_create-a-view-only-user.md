Create a View Only user
Supported in:
Google secops
SOAR
The administrator can add view only users to the Google Security Operations platform. This can be useful when you want somebody in your organization or an MSSP customer to be aware of information pertaining to specific places in the platform but without having the ability to change them.
The procedure to add these users is the same as that of adding regular users:
Purchase the license for the required number of users with your Account Manager.
Create a new view only user group or use the predefined view only group.
Create a new user.
Purchase a view only license
Arrange a license for the required number of view only users.
Go to
Settings
>
Organizations
>
License Management
to view the details.
Set up a permission group
Go to
Settings
>
Organization
>
Permissions
.
Click the predefined permissions group called
View Only
or create a new one.
Select the
landing page
from the list.
Click
Save
.
Create a view only user - for SOAR standalone users only
Go to
Settings
>
Organization
>
User Management
.
Click
add
Add
to open the
Add User
dialog.
In the
License Type
field, select the
View Only
type.
In the
Permission Group
list, select the
View Only
group (or any new group) that you created for
View Only
users.
Fill out the rest of the fields as required.
Click
Add
. An email invitation is sent automatically to the user. Their status remains
Pending
until they accept the invitation to join Google Security Operations SOAR and create a password.
