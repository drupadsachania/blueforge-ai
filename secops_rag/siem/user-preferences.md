Configure user preferences
Supported in:
Google secops
SOAR
The
User Preferences
dialog is where you manage your
user-related configurations, such as localization, notifications, and 
accessibility. The dialog lets you configure your time zone, date/time formats, 
notification settings, and accessibility options.
To configure user preferences, do the following:
Click your user avatar.
Select
User preferences
from the menu.
In the
User preferences
dialog, access the 
        following options:
Localization
:
Time Zone
: configure your time zone.
Date/Time Format
: set your preferred date and time 
        format.
Notifications
:
Choose how many days of past notifications to show in the
Notifications
panel.
Click the
User Notifications
toggle to the ON or OFF position.
Select the types of notifications you want to receive.
Select how you want to receive notifications (on the platform, by 
          email, or both).
Accessibility
:
Set the duration for how long feedback messages (such as confirmations or errors)
          remain visible on the platform before they disappear.
