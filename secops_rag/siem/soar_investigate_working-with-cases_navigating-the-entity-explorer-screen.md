Navigate the Entity Explorer page
Supported in:
Google secops
SOAR
This document explains how to use the
Entity Explorer
page in Google Security Operations.
Entities are the core components of security events. On the
Cases
page, you can click an entity to open the
Entity Explorer
page, where you can review detailed insights and contextual data related to that entity.
The
Entity Explorer
page gives you insights into:
The entity's involvement in cases over the past 90 days. This data might differ
from what's shown in the side panel. For example, a URL flagged
as
suspicious = true
and highlighted in red 
in the side panel, might appear as
false
, depending on how it's
categorized in other cases.
Basic information about the entity, along with enriched information from associated 
cases.
Linked entities, such as related users or IP addresses.
The frequency of each case type that involves the entity.
A list of relevant log entries.

or
