Supported in:
Google secops
SOAR
This document explains how to roll back response integrations to a previously
installed version. This is useful if a new version introduces unexpected
behavior or if you need to revert custom code changes.
Roll back to a previous version
To roll back to a previously installed version, follow these steps:
Go to
Content Hub
and click the
Response Integrations
tab.
Select the required response integration.
Click
Details
.
In the side drawer, click the
Versions
tab.
Locate the
Last Installed
section and click
Rollback
.
In the confirmation dialog, click
Confirm
.
Choose whether to override the existing ontology. For more information 
    on managing ontologies, see
Ontology override
.
Click
Confirm
. The system uninstalls the current
    version and restores the previous version.
