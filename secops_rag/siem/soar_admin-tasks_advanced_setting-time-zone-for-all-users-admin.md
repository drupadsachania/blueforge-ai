Customize your time zone for all users
Supported in:
Google secops
SOAR
In the
Localization
settings, administrators can set the default time
zone and date/time format for all platform users. These changes take effect immediately after saving.
While administrators can set the default time zone, individual users can
  customize these settings to match their own preferences. For more information, see
Configure user preferences
.
To customize these settings, follow these steps:
Go to
Settings
>
Advanced
>
Localization
.
Select the time zone.
Select
Date Format
and
Time Format
to configure the date and time formats.
Click
Save
.
