Create an agent using Docker
Supported in:
Google secops
SOAR
This document outlines the procedure to deploy a Remote Agent using the platform's
Docker installation wizard. This method automates the container setup process on
the host machine.
In the Google Security Operations platform, go to
SOAR Settings
>
Advanced
>
Remote Agents
.
On the
Remote Agents
page, click
add
Add
>
Docker installation
.
In the
Name
field, enter an agent name.
Select an environment for the agent.
Follow all steps in the wizard to complete the deployment.
Click
Next
to perform a connectivity check with your new agent. If the check is successful, the agent is added. If the check fails, a prompt appears with helpful links on how to troubleshoot the issue.
Click
Done
. The agent is now added.
