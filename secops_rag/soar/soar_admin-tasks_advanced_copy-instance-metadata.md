Allow Support to troubleshoot SOAR functionality
Supported in:
Google secops
SOAR
You can authorize Google Support to replicate your instance's runtime configuration to troubleshoot SOAR functionality issues without requiring a live screen-share session.
To provide the necessary instance data, do the following:
Go to the question mark icon at the top of the platform.
Select
Copy Instance Metadata
.
Send the resulting JSON file to your Support representative.
