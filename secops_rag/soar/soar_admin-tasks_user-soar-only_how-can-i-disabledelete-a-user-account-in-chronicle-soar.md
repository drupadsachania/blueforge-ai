Disable or delete a user account
Supported in:
Google secops
SOAR
When you delete a user account, all open cases assigned to this user are 
  automatically assigned to the
Default Role
.
  We strongly recommend that you check the open tasks and pending actions that are 
  assigned to the user and reassign them before deleting the user.
This document explains how to disable or permanently delete a user account from the
User Management
section of the platform's settings.
To disable or delete a user account, follow these steps:
Go to
Settings
>
Organization
>
User Management
.
Double-click to highlight the required user.
Do one of the following:
To
disable
a user account: In the
Edit user
dialog, select the
User Account is Disabled
checkbox and then click
Save
.
To permanently
delete
the user: In the
Edit user
dialog, click
delete
Delete
and then click
Confirm
.
