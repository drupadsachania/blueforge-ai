Types of users for Google SecOps SOAR
Supported in:
Google secops
SOAR
There are five types of users:
Standard Users
: Regular users who can be given various permissions and have edit rights to the platform as you defined them.
View-Only Users
: Users who can only view certain parts of the platform depending on the permissions you assign them. They require a special license. Learn how to
create a view-only user
.
Collaborator Users
: Have edit or view permissions for specific platform modules. For details, see
Benefits of adding a collaborator user
.
Managed Users
: Have full case management permissions, letting them participate in a hybrid SOC model alongside a Managed Security Service Provider (MSSP).
For more information on Managed Users, click
Create a Managed and Managed-Plus user
.
Managed-Plus Users
: Have the same case management permissions as managed users, with the additional ability to build and edit playbooks within their own environment. For more information on Managed-Plus users, click
Create a Managed and Managed-Plus user
.
