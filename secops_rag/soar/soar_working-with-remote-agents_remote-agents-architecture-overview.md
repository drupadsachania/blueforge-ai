Remote Agent architecture overview
Supported in:
Google secops
SOAR
The
Remote Agent
is a lightweight application installed on a customer's
  remote environment. It connects to the Google Security Operations platform, giving
  your security team access to execute actions and playbooks on the remote environment.
The agent communicates with Google SecOps through the
     Publisher, a managed proxy server. This
     connection automatically syncs the agent with Google SecOps
     platform, fetching and executing tasks from configured connectors and integrations.
