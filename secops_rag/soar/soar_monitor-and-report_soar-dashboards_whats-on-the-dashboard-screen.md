Explore the SOAR Dashboards page
Supported in:
Google secops
SOAR
On the
SOAR Dashboards
page, you can share your dashboards with other
  users, delete dashboards you no longer need, refresh data on the dashboard at
  any time, save dashboards as report templates, and import or export dashboards
  to another platform. You can also set a dashboard as your default.
Work with SOAR dashboard features
Use SOAR dashboards to filter, share, export, and save visualizations of your
security operations data. This page outlines the key features and how to use them.
Select timeframe and environments
Use the controls on the
SOAR Dashboards
page to select the time range and environments (select one or more, or choose
All
) for the data displayed on the dashboard.
Create a new custom dashboard
While the
SOAR Dashboards
page includes a predefined dashboard, you can
  also create and share new custom dashboards.
To create a new custom dashboard, follow these steps:
Go to
Dashboards & Reports
>
SOAR Dashboards
.
Click
keyboard_arrow_down
View all dashboards
and select
add
Create New Dashboard
.
Enter the name for the new dashboard and click
Create
.
Share your dashboard
You can share your dashboard with other users on the system:
Select the dashboard you want to share and click
format_list_bulleted
Bullet list
.
Click
share
Share dashboard with others
.
On the
Share dashboard with others
dialog, select team members or roles to share with.
Click
Apply
.
Delete a dashboard
To delete a dashboard, follow these steps:
Open the dashboard you want to delete and click
delete
Delete
.
In the
Delete Dashboard
dialog, click
Confirm
to delete the selected dashboard.
Save a dashboard as a report template
Select the dashboard you want to save as a report template and click
format_list_bulleted
Bullet list
>
Save as a report template
.
On the
Save dashboard as a report template
dialog, select
Category
from the list, and enter a
Name
.
Click
Save
. This report appears on the
SOAR Reports
page.
