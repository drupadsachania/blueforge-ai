PCI curated dashboards overview
Supported in:
Google secops
SIEM
This document describes Payment Card Industry (PCI) dashboards that provide a
centralized view of key metrics and information on an organization's PCI compliance.
These dashboards help you take immediate action in response to security threats.
The PCI dashboards use reference lists. For more information about reference lists,
see
Reference list
.
The following PCI dashboards are available as curated dashboards:

Anti-malware dashboard
The Anti-malware dashboard provides an overview of PCI assets and active
threat detections for enhanced visibility and control. This dashboard displays
the number of active Endpoint Detection and Response (EDR) sensors for device
compliance. It also categorizes alerts and events by severity, including
allowed and blocked detections, using the
PCI_Assets
reference list.
Network dashboard
The Network dashboard provides insights into network performance, activity,
and traffic trends, helping you understand network behavior, maintain security,
and monitor PCI Data Security Standards (DSS) compliance. It provides a comprehensive overview of
network activity and events, with filtering options to refine data analysis by
time period and data source. This dashboard uses the
PCI_Network_Ranges
reference list.
Alert overview dashboard
The Alert overview dashboard lets you monitor, manage, and respond to security
events, helping you monitor PCI DSS compliance for safeguarding cardholder data.
It provides a real-time, consolidated view of alerts and insights into potential
security breaches and PCI-related events. This dashboard
uses the
PCI_Assets
reference list.
Identity and access dashboard
The Identity and access dashboard protects cardholder data and simplifies
compliance reporting. It provides a real-time overview of Identity and Access Management
activity, including key metrics like user access permissions, password changes,
and authentication events. This ensures access to sensitive data and network resources aligns with PCI DSS standards.
This dashboard uses the
PCI_Assets
reference list.
Monitoring and testing dashboard
The Monitoring and testing dashboard lets you monitor key activities and events,
including critical incidents, failed events, and administrative access. The dashboard also helps monitor account
activity and highlights metrics like audit policy changes and top access by source
to manage threats, providing insights into failed events and audit policy changes.
This dashboard uses the
PCI_Assets
reference list.
Vulnerability management dashboard
The Vulnerability management dashboard helps you monitor and track vulnerabilities
in network systems. It provides real-time monitoring, facilitates regular tracking,
streamlines compliance reporting, and identifies, assesses, and manages vulnerabilities
to maintain secure payment environments. This dashboard uses the
PCI_Assets
and
PCI_Network_Ranges
reference lists.
Patch management dashboard
The Patch management dashboard provides an overview of the patch management
lifecycle, helping you monitor and manage system updates for security and efficiency.
It lets you track successful and failed installations, monitor download statuses,
and identify systems with missed or available updates. This dashboard uses the
Patch_Updates
and
PCI_Assets
reference lists.
Secure configurations dashboard
The Secure configurations dashboard displays real-time monitoring and tracking
of key metrics, helping identify and mitigate potential risks by maintaining
critical configurations. It provides insights into trends, patterns, and potential
gaps, letting you manage configurations from initiation through enforcement.
The dashboard proactively tracks policy changes, failed logins, and default
account usage to identify weaknesses. It also sends real-time alerts for critical
updates, policy changes, and failed logins, letting you take quick action on
issues like failed updates or unauthorized login attempts.
This dashboard uses the
PCI_Assets
and
Default_Users
reference lists.
Data encryption dashboard
The Data encryption dashboard helps you secure sensitive data, identify
vulnerabilities, and maintain PCI compliance through real-time and historical
analysis of data encryption. It provides a high-level summary of data encryption
activity and key management practices, including a number of key operations,
deletions, and decryption results. The dashboard distinguishes between secure
and potentially vulnerable encryption practices. This dashboard uses the
PCI_Network_Ranges
reference list.
EDR alert overview dashboard
The EDR alert overview dashboard provides security teams with a complete picture
of assets, events, and active threat detections, enabling them to quickly detect,
investigate, and respond to potential threats. It summarizes overall EDR activity
and the number of events and alerts. It prioritizes investigations by sorting alerts
and events by severity and listing the top ten alerts, hosts, techniques, and tactics.
It also helps evaluate the effectiveness of the EDR solution.
This dashboard uses the
PCI_Assets
the reference list.
