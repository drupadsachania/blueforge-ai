Move a case to a new environment
Supported in:
Google secops
SOAR
This document explains how to move a case from one environment to another, or duplicate it
across environments. This is useful if your organization uses multiple environments and you need to transfer or share cases between them.
To move or duplicate a case to another environment, follow these steps:
Go to the
Cases
page.
Open the case you want to move.
In the case view, click
Environment
and then select the target environment.
In the
New Environment
dialog, select one of the following options:
Confirm
: Creates a copy of the case in the selected environment. The new case has a different case ID.
Close current case
(checkbox)
>
Confirm
: Closes the case in the current environment and creates a new case with a different case ID in the selected environment.
