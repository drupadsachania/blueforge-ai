Manage password settings
Supported in:
SOAR
This document explains how to create Google Security Operations SOAR passwords
and manage their settings. For security, we recommend using a strong password that meets the following requirements:
A minimum of 10 characters
At least one capital letter
At least one special character
Change your password
To change your password, follow these steps:
Click your user avatar.
In the menu, select
Change Password
.
Enforce mandatory password changes
To safeguard your data, password changes are required every 90 days. The 
following protocol is in place:
Passwords expire after 90 days.
Users receive the following notifications when their password is nearing expiration:
14 days before password expiry
7 days before password expiry
2 days before password expiry
1 day before password expiry
An account locks if a password expires or if a user attempts multiple incorrect logins.
Administrators can re-enable locked user accounts, and the password is then valid for another 90 days.
