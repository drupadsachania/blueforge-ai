Use the Playbook Navigator
Supported in:
Google secops
SOAR
This document describes how to use the Playbook Navigator to efficiently search and navigate steps within your playbooks. The Playbook Navigator provides a searchable, filterable list of all actions and flows, letting you quickly locate and jump to any step without manually scrolling through the entire playbook visualization.
Access the Playbook Navigator
Follow these steps to access and use the Playbook Navigator:
Go to the
Playbooks
page and select a playbook.
Confirm that the Playbook Simulator is disabled
Click the
Playbook Navigator
icon to open the navigator side drawer. (Typically, the icon is represented as a search or navigation symbol.)
Search and filter steps
Once you click the Playbook Navigator icon, the Playbook Navigator side drawer opens and shows you all the steps (actions and flows) in the playbook. Click the step you want to find quickly inside the playbook. You can use the search bar to find a step by name or use the filters for more granular control:
Filter
Description
Integration Name
Filter by the specific integration used in the step.
Step Type
Filter results by the type of step (All, Action, Condition, or Block).
Skip if Failed
Filter by the step's error handling setting (All, Stop playbook, or Skip step).
Step Mode
Filter by the step's execution mode (All, Automatic, or Manual).
To instantly locate a step, click its name in the Playbook Navigator side drawer. The Playbook Designer will automatically scroll and highlight that step.
