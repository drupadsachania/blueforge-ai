Add or delete case stages
Supported in:
Google secops
SOAR
This document explains how to customize case stages to match your organization's case 
management stages.
Add a new case stage
To add a new case stage, follow these steps:
Go to
SOAR Settings
>
Case Data
>
Case 
stages
.
Click
add
Add Stage
.
In the
Add Case Stage
dialog, enter the relevant information for the new case stage.
Click
Create
.
Delete a case stage
To delete a case stage, do the following:
Go to
SOAR Settings
>
Case Data
>
Case 
Stages
.
Select the case stage you want to delete.
Click
edit
Edit Stage
(or double-click the case stage) to open the
Edit Case Stage
dialog.
In the
Edit Case Stage
dialog, click
delete
Delete
.
In the
Confirmation
dialog, click
Yes
to remove the case 
stage.
